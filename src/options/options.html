pAIdegua — Modelos de Minuta

Configure a pasta com seus modelos de minutas. A extensão usa esses modelos como referência para gerar peças no estilo que você já adota, preservando estrutura e linguagem.

Pasta de modelos

Selecione a pasta raiz onde estão suas minutas-modelo. Subpastas são lidas recursivamente — você pode organizar por categoria (ex.: procedente, improcedente, despachos). Formatos aceitos: .docx, .pdf, .txt, .md.

Aviso para usuários do Firefox: este recurso depende da File System Access API, ainda não suportada pelo Firefox. Para configurar modelos de minuta, utilize o Google Chrome ou o Microsoft Edge. As demais funcionalidades do pAIdegua (geração de minutas, anonimização, extração de documentos) funcionam normalmente no Firefox.

    Como funciona

    • Os arquivos ficam armazenados localmente no navegador (IndexedDB), sem sair da sua estação.
    • Apenas o texto é indexado — sua pasta original não é modificada.
    • Você pode optar por gerar minutas com base nos modelos ou do zero a cada uso, diretamente na barra lateral do PJe.
    • Reindexar reprocessa toda a pasta — útil quando você adicionar, editar ou remover modelos.
